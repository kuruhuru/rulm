Составь набор из {{num_tasks}} разных заданий для дообучения языковой модели:

1. Делай задания максимально разными: по типу, по используемому глаголу, по формулировке.
2. Задания должны быть выполнимы языковой моделью, работающей только с текстами и не имеющей доступа ко внешнему миру.
3. Используй хороший русский.
4. Делай задания в одно или два предложения.
5. Генерируй подходящие реалистичные входные данные, не используй общие термина типа \"Имя человека\" вместо реального имени.
6. Задание может быть без входных данных, в таком случае используй токен <noinput> вместо них.
7. На выходе сгенерируй подходящий не слишком длинный ответ.
8. Следуй тому же шаблону, который приведен в примерах, разделяй задания с помощью ###. Это важно!
